# WsNum

Программа создана с целью облегчить некоторый функционал моей работы. Позже будут добвлены инструкции к применению. На данный момент существует сырая версия. Надеюсь фраза "Все временное становится постоянным" не сыграет здесь своей роли. 

Программа написана с помощью ИИ, так как С++ я еще не изучал. 

Инструкция к использованию. 
1. Скопировать номер телефона в буфер обмена в стандартных форматах. 
2. Нажать Ctrl + D, после номер будет или вставлен в текстовое поле, или сохранен в буфере обмена.
3. Во втором случае требуется вставка. То есть если программа не позволяет вставить номер в поле, нужно нажать последовательно (Ctrl + D) + (Ctrl + V).
4. Существует анализ ошибок, при неправильном использовании программы появится окно с информацией об ошибке.
